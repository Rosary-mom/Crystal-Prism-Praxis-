# Crystal Prism Praxis

CRYSTAL PRISM v45 SPARC validation and Mars Mission triple-engine hub.

|------|-------------|
| `index.html` | Enhanced Mars Mission cockpit — DWARF · Crystal Prism Vortex · Alcubierre Warp |
| `mars-mission-classic.html` | Live mirror of [blizzwar.vercel.app](https://blizzwar.vercel.app/) |

| `crystal_prism_v45.py` | Full 175-galaxy SPARC catalog validation |

## Quick start

### Web (local)

```bash
python -m http.server 8080
# open http://localhost:8080/index.html
```

### SPARC validation

```bash
pip install -r requirements.txt
python crystal_prism_v45.py
```

## Deploy

Static site — deploy root to Vercel or GitHub Pages. `vercel.json` routes `/` to `index.html`.
